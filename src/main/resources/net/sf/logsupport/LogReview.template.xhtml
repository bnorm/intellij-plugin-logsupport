Notes:
  • This is an editable review of a collection of log messages that exist inside the project. When editing messages, please be careful not to modify or remove the markers for constant value "#?#" or break "#/#". If the markers are missing the changes will not match the corresponding sources and integrations of the review will fail on the message.
  • Adjustments to this document can be made with any XML/XHTML aware editor, including favourite word processors as long as they do not modify the general document structure and are able to store the result as XHTML document. Any visual formatting applied to the messages will be stripped when integrating the changes into the project.
    When the document structure was modified or the resulting document is not valid XHTML, it will be very likely not possible to integrate the changes into the project.
  • Depending on the settings that were applied when this review was created, the collection of messages may be equal to all the log messages that exist inside the project.
  • It's not possible to change the log level or source file by editing the values inside this review. Changes to these columns are ignored.
